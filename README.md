## âš™ï¸ Installation & Setup

1. Install **XAMPP** or **WAMP** server.
2. Clone or download this repository.
3. Move the project folder to:

   * `htdocs` (XAMPP) or
   * `www` (WAMP)
4. Start **Apache** and **MySQL** from the control panel.
5. Create a database in phpMyAdmin.
6. Import the provided SQL file (if available) or create an `employee` table manually.
7. Update database credentials in `db.php`.
8. Open the browser and visit:

   ```
   http://localhost/project-folder-name/
   ```
